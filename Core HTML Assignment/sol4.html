List of Tags

  1. Span: The span tag in HTML is an inline container used to mark up a part of a text, or a part of a document. It can be used to group inline-elements in a document.
  2. Img: The img tag in HTML is used to embed an image in a web page. Images are not technically inserted into a web page
  3. a: The a tag in HTML is used to create hyperlinks, which can link from one page to another, or link sections within a document
  4. ol: ol: The ol tag in HTML is used to define an ordered list. An ordered list can be numerical or alphabetical
  5. form: The form tag in HTML is used to create an HTML form for user input. It can contain input elements like text fields, checkboxes, radio-buttons, submit buttons and more
  6. p: The p tag in HTML is used to define a paragraph. It automatically creates some space before and after itself, and also respects all whitespace and line breaks within the text
